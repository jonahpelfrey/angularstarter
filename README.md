# Angular Starter Application
### Installation and Serving
+ Clone this repo onto your machine
+ Run **npm install** to install node modules
+ Run **node server** to start a local instance of the app on port 8000
+ Point your browswer to http://localhost:8000/


### Information
+ This template uses bootstrap CSS (https://www.w3schools.com/bootstrap)
+ Server uses ExpressJS for backend routing (http://expressjs.com/en/guide/routing.html)
+ You will need to install NodeJS (https://nodejs.org/en/download/)
+ Angular Documentation (https://angularjs.org/)
